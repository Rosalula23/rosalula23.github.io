< body >
< body > marks which text makes up the body of the document
This is an example of body. It doesn't look much different than the list items or other text. But the < body > tag was used to define the content.
< ul>
< ul > starts an unordered list. An unordered list is marked with small black circles by default.
After using < ul >, list items are added using < li >item< /li>.
  • Item 1
  • Item 2
  • Item 3
< ol >
< ol > stars an ordered list. This list is marked with numbers by default
Like with < ul >, list items are added using < li >item< /li>.
  1. Item 1
  2. Item 2
  3. Item 3
< a href >
< a href > allows you to insert a hyperlink.
This looks like < a href="url">link text< /a>
Here's a better explanation
< img >
< img > allows you to insert an image.
This looks like < img src="filepath" alt="Name" >
You can also specify the size of the image
< img src="filepath" alt="name" width="X" height="X">
< strong >
< strong > indicates text which is very important.
This is very important!
This text is bold.
These look the same, but are different for those using accessibilty aids and those poking through the code.
& nbsp;
This addition in a line of text inputs an extra space between words without creating a new line.
This looks like < p > text text & nbsp; text < /p >
